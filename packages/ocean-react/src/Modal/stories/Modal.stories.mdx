# Modal API

The API documentation of the Modal React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { Modal } from '@useblu/ocean-react';
```

## Usage

```jsx
<Modal isOpen={false}>Hello There!</Modal>
```

## Props

<ArgsTable of="." />

Any other props supplied will be provided to the ReactModal element (native element).

## CSS

| Global class               | Description                             |
| -------------------------- | --------------------------------------- |
| .ods-checkbox\_\_root      | Styles applied to the root element.     |
| .ods-checkbox              | Styles applied to the checkbox element. |
| .ods-checkbox\_\_checkmark | Styles applied to the span element.     |
| .ods-checkbox\_\_label     | Styles applied to the span element.     |

If that's not sufficient, you can check the [implementation of the component](https://github.com/Pagnet/ocean-ds-web/blob/master/src/Modal/Modal.tsx) for more detail.
