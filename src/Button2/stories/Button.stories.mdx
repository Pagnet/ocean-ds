# Button API

The API documentation of the Button React component. Learn more about the props and the CSS customization points.

```javascript

```

## Props

<Props component={Button} parameters />

The ref is forwarded to the root element.

Any other props supplied will be provided to the root element (native element).

## CSS

| Global class         | Description                          |
| -------------------- | ------------------------------------ |
| .ods-btn             | Styles applied to the root element.  |
| .ods-btn--primary    | The color of the primary component   |
| .ods-btn--secondary  | The color of the secondary component |
| .ods-btn--ghost      | The color of the ghost component     |
| .ods-btn--inverse    | The color of the inverse component   |
| .ods-btn--[sm,md,lg] | The width set to the component       |
| .ods-btn--full-width | Full width to the component          |

If that's not sufficient, you can check the [implementation of the component](https://github.com/Pagnet/ocean-ds-web/blob/master/src/Button/Button.tsx) for more detail.
