## CSS

| Global class                     | Description                                            |
| -------------------------------- | ------------------------------------------------------ |
| .ods-input\_\_root               | Styles applied to the div element.                     |
| .ods-input                       | Styles applied to the input element.                   |
| .ods-input--error                | Styles applied to the input element if error={true}.   |
| .ods-input--blocked              | Styles applied to the input element if blocked={true}. |
| .ods-input\_\_label              | Styles applied to the label element.                   |
| .ods-input\_\_helper-text        | Styles applied to the p element.                       |
| .ods-input\_\_helper-text--error | Styles applied to the p element if error={true}.       |

If that's not sufficient, you can check the [implementation of the component](https://github.com/Pagnet/ocean-ds-web/blob/master/src/Input/Input.tsx) for more detail.
